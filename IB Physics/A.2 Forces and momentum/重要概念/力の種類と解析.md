# 力の解析：分類と運動法則の適用

![[力の種類と解析.png]]

> [!ABSTRACT] 概要
> 力（Forces）の解析は、[[ニュートンの運動法則]]を適用して物体の挙動を理解するための核心的なプロセスです。提供された資料に基づき、力の「種類（分類）」と、それらをどのように扱って「解析」するかについてまとめました。

---

## 1. 力の分類：場の力と接触力

解析の第一歩は、見落としなく力を特定することです。力は大きく以下の2つに分類されます。

* **場の力 (Field Forces):**
  物体が何かに触れていなくても、距離を隔てて働く力です。
    * **重力 ($F_g$ または $W$):** 地球が物体を引く力。質量 $m$ の物体に対し、鉛直下向きに $F_g = mg$ の大きさを持ちます（$g$ は重力加速度 $9.81\text{ m/s}^2$）。
    * その他、**電気力** ($F_e$) や **磁気力** ($F_m$) もここに含まれます。

* **接触力 (Contact Forces):**
  物体が他の物体や流体と直接接触している場合にのみ働く力です。「タッチの定理」として、触れているもの一つにつき力が働くと覚えると見落としが減ります。

## 2. 主な力の種類と性質

![[力の種類.png]]

解析において頻出する具体的な力は以下の通りです。

| 力の種類         | 記号・公式                | 特徴                                              |
| :----------- | :------------------- | :---------------------------------------------- |
| **[[垂直抗力]]** | $F_N$ または $R$        | 面が物体を垂直に押す力です。接触面に対して直角に働きます。                   |
| **[[張力]]**   | $T$ または $F_t$        | 糸やロープが物体を引く力です。糸の方向に働きます。                       |
| **[[摩擦力]]**  | $F_f$                | 面に平行に働き、相対運動を妨げる向きに作用します。                       |
| 　静止摩擦        | $F_f \leq \mu_s F_N$ | 物体が静止している間、外力と釣り合う大きさで働きます。最大値は $\mu_s F_N$ です。 |
| 　動摩擦         | $F_f = \mu_d F_N$    | 物体が滑っている間に働き、一定の値をとります（面積や速度に依存しません）。           |
| **[[弾性力]]**  | $F_H = -kx$          | バネが自然長に戻ろうとする力（フックの法則）。変位 $x$ に比例し、向きは変位と逆です。   |
| **[[空気抵抗]]** | $F_d$                | 流体中を動く物体が受ける抵抗力。速度が増すと大きくなり、重力と釣り合うと終端速度に達します。  |
| **[[浮力]]**   | $F_b = \rho Vg$      | 流体中の物体に働く上向きの力。押しのけた流体の重さに等しい（アルキメデスの原理）。       |

## 3. 力の解析手順（Analysis）

特定した力を用いて運動を解析する手順は以下の通りです。

### ① [[自由体図]] (Free-Body Diagram) の作成
* 着目する物体を「点」として扱い、その物体が受ける力（重力＋接触力）のみを矢印で描きます。
 > [!IMPORTANT] 注意
  > 作用・反作用の法則（第三法則）に基づき、その物体が「他へ及ぼす力」は描いてはいけません。

![[自由物体図.png]]
### ② 成分への分解 (Resolution of Forces)
* 斜面上の運動などでは、力を「運動方向」と「それに垂直な方向」の成分に分解します。
* **例：斜面上の箱**
  重力 $mg$ を斜面に平行な成分 ($mg \sin \theta$) と垂直な成分 ($mg \cos \theta$) に分解します。

### ③ 方程式の立式
* **平衡状態（静止または等速直線運動）:** 合力がゼロになります。
  $$\sum F = 0$$
  これを利用して、未知の力（張力や垂直抗力など）を計算します。
* **加速状態:** ニュートンの第二法則([[運動方程式]])を適用します。
  $$F_{net} = ma$$
  合力（不釣り合いな力）が加速度の原因となります。

### ④ 円運動の場合
* 中心に向かう力を正として合力を計算します。この合力が **[[向心力]]** ($mv^2 / r$) となります。
> [!INFO] ポイント
  > 向心力という新しい力があるわけではなく、張力や摩擦力などがその役割を果たします。

> [!IMPORTANT] まとめ
