# エネルギー源（Energy Sources）の概念と効率

> [!ABSTRACT] 概要
> 本ノートでは、機械やシステムが仕事を行うために必要な「エネルギー源」について、物理学的な観点からその種類、定量化の指標（エネルギー密度）、および変換効率についてまとめます。特に、エネルギーの質を比較するための「エネルギー密度」や、流れを可視化する「サンキーダイアグラム」の重要性に焦点を当てます。


![[エネルギー源.png]]
## 1. 主なエネルギー源の種類

仕事をするためのエネルギーは、様々な形態で貯蔵されています。これらは最終的に熱や運動エネルギーへと変換されます。

* **燃料（化学エネルギー）**: ガソリン、ガス、石炭、食物（砂糖、油脂）など。燃焼や化学反応を介して仕事に変換されます。
* **太陽エネルギー**: 地球上のほぼすべてのエネルギー循環の根本的な源です。
* **水力（位置エネルギー）**: 重力によって高所に蓄えられたエネルギー。太陽熱による蒸発と降雨のサイクル（水循環）に依存するため、間接的な太陽エネルギーと言えます。
* **その他**: 原子力、電気、風力など。

---

## 2. エネルギー密度 (Energy Density)

燃料の「質」や「輸送効率」を評価するための物理学的な指標です。

> [!IMPORTANT] 定義と指標
> **エネルギー密度**：燃料の単位体積あたりに含まれるエネルギー量 ($J/m^3$ または $MJ/L$)。
> **比エネルギー（Specific Energy）**: 単位質量あたりのエネルギー ($J/kg$)。
> 
> 両者の関係は以下の通りです：
> $$\text{エネルギー密度} = \text{比エネルギー} \times \text{物質の密度}$$

### 燃料別の比較（概算値）
| 燃料種別 | エネルギー密度 ($MJ/L$) | 備考 |
| :--- | :--- | :--- |
| **石炭** | $72 \, MJ/L$ | 非常に高い密度 |
| **原油** | $37 \, MJ/L$ | |
| **ガソリン・ディーゼル** | 約 $36 \, MJ/L$ | |
| **木材** | $3 \, MJ/L$ | 密度が低い |

## 3. エネルギーの変換と効率

エネルギーは変換の過程で必ず一部が「損失」として散逸します。

### サンキーダイアグラム (Sankey Diagram)
エネルギーの流れと損失を視覚化するツールです。
* **矢印の幅**: エネルギーの電力量に比例。
* **保存則**: $\text{入力の幅} = \text{有用な出力} + \text{損失（熱など）}$



### 効率 ($\eta$) の計算
投入されたエネルギーのうち、目的の仕事に変換された割合を指します。

$$
\eta = \frac{\text{有用な出力エネルギー}}{\text{総入力エネルギー}} = \frac{\text{有用な出力仕事率}}{\text{総入力仕事率}}
$$

> [!INFO] 具体例
> 内燃機関（ガソリンエンジン）の場合、燃料の多くが摩擦や熱として失われるため、効率は $100\%$ 未満（例：$28\%$ 程度）に留まります。

## 4. 具体例：水力発電の計算

水力発電は、重力による位置エネルギーを電気エネルギーへ変換するプロセスです。

### 計算モデル
湖（貯水池）にある水がタービンに向かって落下する場合、利用可能な位置エネルギー $E_p$ は次式で表されます。

$$
E_p = mgh
$$

* $m$: 水の質量
* $g$: 重力加速度
* $h$: 平均落下高さ

### 仕事率（Power）の導出
単位時間あたりの発電出力（ワット）を求めるには、1秒あたりの流量（質量流量）を用いて計算します。
